# managethedepots

Manage Depots allows you to control building access and visibility for your team. This feature ensures dispatchers and employees see only the relevant locations for their daily operations. You will achieve a streamlined workflow by mapping specific buildings to the right users.

### Getting Started

Prerequisites and initial setup steps:
*   Administrative access to the **Configuration** page.
*   Active user accounts created in the system.

1. Open the **Configuration** page.

![Frame at 0:10](../images/managethedepots-managethedepots_timestamp_0_to_10.png "Configuration – The main settings menu where user management begins.")

2. Select **Manage Users**.

![Frame at 0:15](../images/managethedepots-managethedepots_timestamp_0_to_15.png "Manage Users – The interface for editing employee profiles and permissions.")

### Feature Overview

*   **Manage the Reports**: This option under the **Delivery** section allows you to view and organize building assignments.

![Frame at 0:41](../images/managethedepots-managethedepots_timestamp_0_to_41.png "Manage the Reports – The menu item located under the Delivery category.")

*   **Roles and Rights**: A sub-menu within a user profile where you toggle specific feature permissions.

![Frame at 0:20](../images/managethedepots-managethedepots_timestamp_0_to_20.png "Roles and Rights – Settings page used to enable specific user capabilities.")

*   **Location Dropdown**: A selection menu in the user profile used to map a specific building to an employee.

![Frame at 1:05](../images/managethedepots-managethedepots_timestamp_1_to_05.png "Location Dropdown – A field used to assign a specific building to a user profile.")

*   **Quick Access Building Selector**: A tool in the top right corner for rapid building switching.

![Frame at 1:18](../images/managethedepots-managethedepots_timestamp_1_to_18.png "Top Right Corner – A shortcut for selecting and switching active buildings.")

### How To: Enable Depot Management

1. Navigate to the **Configuration** page and click **Manage Users**.

![Flow 0:10 to 0:18](../images/managethedepots-managethedepots_timestamp_0_to_10_to_0_to_18.gif "Navigation – Moving from the configuration page to the user list.")

2. Click on a specific **User**.

3. Select **Roles and Rights**.

4. Enable the **Manage the Reports** option.

![Frame at 0:25](../images/managethedepots-managethedepots_timestamp_0_to_25.png "Manage the Reports Toggle – Activating the depot management capability for a user.")

5. Click the **Left Arrow** to return to the main menu.

![Flow 0:27 to 0:35](../images/managethedepots-managethedepots_timestamp_0_to_27_to_0_to_35.gif "Go Back – Using the navigation arrow to return to the Delivery section.")

### How To: Map a Building to a User

1. Select **Manage Users** and click on the desired **User**.

2. Locate the **Location Dropdown** menu.

![Frame at 1:07](../images/managethedepots-managethedepots_timestamp_1_to_07.png "Location Dropdown – Finding the building assignment field within user settings.")

3. Select the building from the list.

4. Refresh the page to confirm the mapping reflects in the **Manage Users** table.

![Flow 1:22 to 1:41](../images/managethedepots-managethedepots_timestamp_1_to_22_to_1_to_41.gif "Refresh and Verify – Updating the page to see the new building assignment in the table.")

### Productivity Tips

*   💡 **Quick Access**: Use the top right corner selector to switch buildings instantly without entering user settings.
*   ⚠️ **Mobile Sync**: Changes made in the mobile application will automatically update the back office settings.
