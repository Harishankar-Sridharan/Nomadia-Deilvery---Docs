# iframeconfiguration

Configure iFrame settings for subcontractors within Nomadia Delivery to embed external mission data. This feature allows you to customize the display of external URLs for specific users. You will achieve a personalized viewing experience for each subcontractor profile.

### Getting Started

*   Access to the Nomadia Delivery dashboard.
*   A valid machine URL or mission ID.

1. Navigate to the **Subcontractors** menu.

![Frame at 0:10](../images/iframeconfiguration-iframeconfiguration_timestamp_0_to_10.png "Subcontractors – The main dashboard view showing the user list.")

2. Locate the list of users on the screen.

### Feature Overview

*   **Pencil icon**: Opens the editing interface for a selected subcontractor.

![Frame at 0:18](../images/iframeconfiguration-iframeconfiguration_timestamp_0_to_18.png "Pencil icon – The edit button located next to a specific user entry.")

*   **iFrame URL**: Field used to input the machine URL or mission ID.

![Frame at 0:25](../images/iframeconfiguration-iframeconfiguration_timestamp_0_to_25.png "iFrame URL – The input field for embedding external web content.")

*   **iFrame width**: Sets the horizontal dimension of the embedded frame.

![Frame at 0:37](../images/iframeconfiguration-iframeconfiguration_timestamp_0_to_37.png "iFrame width – The numeric field for adjusting the width of the display.")

*   **iFrame height**: Sets the vertical dimension of the embedded frame.

![Frame at 0:43](../images/iframeconfiguration-iframeconfiguration_timestamp_0_to_43.png "iFrame height – The numeric field for adjusting the height of the display.")

*   **Save**: Commits the configuration changes to the subcontractor profile.

![Frame at 0:54](../images/iframeconfiguration-iframeconfiguration_timestamp_0_to_54.png "Save – The action button to finalize and store all modifications.")

### How To: Configure Subcontractor iFrame

1. Go to **Subcontractors**.

![Frame at 0:10](../images/iframeconfiguration-iframeconfiguration_timestamp_0_to_10.png "Subcontractors – Navigation to the main user management screen.")

2. Click the **pencil icon** next to the user.

![Flow 0:10 to 0:18](../images/iframeconfiguration-iframeconfiguration_timestamp_0_to_10_to_0_to_18.gif "Edit User – Transitioning from the user list to the configuration profile.")

3. Replace the existing URL with your machine URL or mission ID.

![Flow 0:18 to 0:30](../images/iframeconfiguration-iframeconfiguration_timestamp_0_to_18_to_0_to_30.gif "URL Entry – Updating the iFrame source with a new mission identifier.")

4. Set the **iFrame width** value.

![Frame at 0:37](../images/iframeconfiguration-iframeconfiguration_timestamp_0_to_37.png "iFrame width – Entering the width specifications in the field.")

5. Set the **iFrame height** value.

![Frame at 0:43](../images/iframeconfiguration-iframeconfiguration_timestamp_0_to_43.png "iFrame height – Entering the height specifications in the field.")

6. Click on **save** to save the modifications.

![Flow 0:49 to 0:59](../images/iframeconfiguration-iframeconfiguration_timestamp_0_to_49_to_0_to_59.gif "Save Success – Finalizing the edit and receiving a modification confirmation popup.")

### Productivity Tips

- 💡 **Modification Confirmation**: Verify the configuration by waiting for the "existing subcontractor has been modified" pop-up.
