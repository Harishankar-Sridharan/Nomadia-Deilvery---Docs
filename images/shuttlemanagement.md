# shuttlemanagement

Shuttle management allows you to organize and name your transportation fleet resources. Use this feature to customize shuttle identifiers and assign them to specific machines. This ensures accurate tracking and planning for your logistics operations.

### Getting Started

*   Access to the **Nomadia Delivery** dashboard.
*   Administrative permissions to edit **Custom Fields** and **Machines**.

1. Log in to your account and navigate to the **Configuration** menu.

![Frame at 0:14](../images/shuttlemanagement-shuttlemanagement_timestamp_0_to_14.png "Configuration Screen – Accessing the system settings menu.")

### Feature Overview

*   **Configuration**: The main menu for system-level settings and data structure.

![Frame at 0:14](../images/shuttlemanagement-shuttlemanagement_timestamp_0_to_14.png "Configuration – The entry point for managing custom settings.")

*   **Custom Fields**: A section to define and edit specific data labels like shuttle names.

![Frame at 0:23](../images/shuttlemanagement-shuttlemanagement_timestamp_0_to_23.png "Custom Fields – The interface for managing organization-specific data fields.")

*   **Pencil Icon**: The edit button used to modify the names of existing shuttles.

![Frame at 0:28](../images/shuttlemanagement-shuttlemanagement_timestamp_0_to_28.png "Edit Icon – Selecting the edit tool for shuttle lists.")

*   **Machines**: The module where all fleet units and hardware are listed and managed.

![Frame at 0:50](../images/shuttlemanagement-shuttlemanagement_timestamp_0_to_50.png "Machines List – View of all registered machines in the system.")

*   **Actions**: A button that opens a menu of mass-management tools for selected items.

![Frame at 1:02](../images/shuttlemanagement-shuttlemanagement_timestamp_1_to_02.png "Actions Menu – Accessing bulk operations for machine management.")

*   **Bulk Edit**: A tool to apply the same change to multiple machines simultaneously.

![Frame at 1:02](../images/shuttlemanagement-shuttlemanagement_timestamp_1_to_02.png "Bulk Edit – The interface for updating multiple records at once.")

*   **Drop**: A toggle or section header used to reveal specific fields like shuttle names.

![Frame at 1:07](../images/shuttlemanagement-shuttlemanagement_timestamp_1_to_07.png "Field Selection – Navigating to the shuttle assignment dropdown.")

### How To: Update Shuttle Names

1. Go to **Configuration**.

![Flow 0:00 to 0:14](../images/shuttlemanagement-shuttlemanagement_timestamp_0_to_00_to_0_to_14.gif "Navigation – Moving from the home screen to Configuration.")

2. Click **Custom Fields**.
3. Click the **Pencil Icon** next to the shuttle field name.

![Frame at 0:23](../images/shuttlemanagement-shuttlemanagement_timestamp_0_to_23.png "Edit Field – Clicking the pencil icon to open the shuttle name list.")

4. Modify the list of shuttle names as needed.
5. Click **Save**.

![Flow 0:28 to 0:38](../images/shuttlemanagement-shuttlemanagement_timestamp_0_to_28_to_0_to_38.gif "Save Action – Finalizing the changes to the shuttle name list.")

### How To: Assign Shuttles to Machines

1. Go to **Machines**.
2. Select the checkbox for the machines you want to update.

![Flow 0:38 to 0:50](../images/shuttlemanagement-shuttlemanagement_timestamp_0_to_38_to_0_to_50.gif "Machine Selection – Picking specific machines for assignment.")

3. Click on **Actions**.
4. Click on **Bulk Edit**.

![Frame at 1:02](../images/shuttlemanagement-shuttlemanagement_timestamp_1_to_02.png "Bulk Edit Menu – Selecting the mass edit tool from Actions.")

5. Click on **Drop**.
6. Click the drop-down menu next to **Shuttle Names**.

![Flow 1:02 to 1:07](../images/shuttlemanagement-shuttlemanagement_timestamp_1_to_02_to_1_to_07.gif "Dropdown Selection – Opening the menu to view available shuttle names.")

7. Select one shuttle name from the list.
8. Click on **Save**.

![Flow 1:12 to 1:20](../images/shuttlemanagement-shuttlemanagement_timestamp_1_to_12_to_1_to_20.gif "Assignment Save – Applying the chosen shuttle to the selected machines.")

### Productivity Tips

- 💡 **Bulk Updates**: Save time by using the **Bulk Edit** feature to assign the same shuttle to multiple machines at once.
