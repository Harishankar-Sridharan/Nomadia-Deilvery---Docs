# Improve the Missions addresses geocoding

### Correct geocoding with a new address

From the Missions page

1. Click **Actions**.
2. Select **Import** from the dropdown menu.
3. Click **Browse (Excel)**.
4. Disable the **Pickup address field**

<figure><img src="../../.gitbook/assets/Nomadia_Layout_V1 (2).png" alt=""><figcaption></figcaption></figure>

5. Select the **Latitude** and **Longitude** from the location indicators

<figure><img src="../../.gitbook/assets/Nomadia_Layout_V1 (3).png" alt=""><figcaption></figcaption></figure>

6. Click on **Validate**

<figure><img src="../../.gitbook/assets/Nomadia_Layout_V1 (4).png" alt=""><figcaption></figcaption></figure>

7. Update the **New address** in the new address field

<figure><img src="../../.gitbook/assets/Nomadia_Layout_V1 (1).png" alt=""><figcaption></figcaption></figure>

#### Select Missions from the map

From the **Map view**.

1. Use the **Selection** button to choose the **selection method** (e.g., Polygon, Circle).

<figure><img src="../../.gitbook/assets/image (171).png" alt=""><figcaption></figcaption></figure>

2. Draw/select the region to highlight missions.

<figure><img src="../../.gitbook/assets/image (172).png" alt=""><figcaption></figcaption></figure>

3. Selected missions will appear in the table view on the left panel.
